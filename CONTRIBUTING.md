# Contributing

Keep changes vertical and tested. Backend changes run through Ruff, mypy, and pytest; frontend
changes run through TypeScript and a production Vite build.

```bash
python -m pip install -e "./backend[dev]"
cd frontend && npm install && cd ..
make check
```

Provider SDK calls stay behind adapters. Preserve citation metadata through every RAG stage, and
never commit real offer documents, secrets, database volumes, vector indexes, or generated legal
guidance.
